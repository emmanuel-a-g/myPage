## Getting Started

To get a local copy up and running follow these simple example steps.
Clone github repo locally and run npm install
If you ecounter any errors, use --legacy-peer-deps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/emmanuel-a-g/myApp.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
4. Enter your Mongodb username and password in `next.config.js`
   ```js
   env : { mongodb_username = 'ENTER YOUR API' }
   ```
